## 10974 모든 순열

- C++ STL 의 next_permutation 을 이용한 예제이다.

```cpp
  // problem url : https://www.acmicpc.net/problem/10974

#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main(){

    int n;
    cin >> n;
    vector <int> a(n);

    for(int i = 0; i < n; i++){
        a[i] = i + 1;
    }

    do{
        for(int i = 0; i < n; i++){
            cout << a[i] << ' ';
        }
        cout << '\n';
    }while(next_permutation(a.begin(), a.end()));

    return 0;
}
```
