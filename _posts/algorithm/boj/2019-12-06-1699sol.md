## 1699 제곱수의 합

- 주어진 자연수 N을 제곱수들의 합으로 표현할 때에 그 항의 __최소개수__ 를 구하는 문제
- ex) 11 = 3^2 + 1^2 + 1^2

- 아이디어는 다음과 같다.
  - D[N] = 자연수 N을 제곱수들의 합으로 표현할 때에 그 항의 __최소 개수__
  - D[N] = MIN (D[N - (i^2)] + 1) ( 1<= i^2 <= N)

```cpp
  // problem url : https://www.acmicpc.net/problem/1699
#include <cstdio>

int d[100001];

int main(){
    int n;
    scanf("%d",&n);

    // d[n] = d[n - i^2] + 1 , * 1 <= i^2 <= N
    for(int i = 1; i <= n; i++){
        d[i] = i;
        for(int j = 1; j * j <= i; j++){
            if(d[i] > d[i - j*j] + 1){
                d[i] = d[i - j*j] + 1;
            }
        }
    }

    printf("%d\n", d[n]);

    return 0;
}
```
