## 11726 2*n 타일링

- 2 * N 직사각형을 1 * 2, 2 * 1 타일로 채우는 방법의 수

- 아이디어는 아래와 같다.
  - D[N] = 2 * N 을 채우는 방법의 수 
  - D[N] 일 때, 1 * 2 와 2 * 1 의 타일을 놓을 수 있는 경우를 고려한다.
  - 점화식은 D[N] = D[N-1] + D[N-2] 이다.

```cpp
  // problem url : https://www.acmicpc.net/problem/11726

#include <iostream>
using namespace std;

int d[1001];

int main(){
    int n;
    cin >> n;

    d[0] = 1;
    d[1] = 1;
    for(int i=2; i <= n; i++){
        d[i] = d[i-1] + d[i-2];
        d[i] %= 10007;
    }

    printf("%d\n",d[n]);
    return 0;
}
```
